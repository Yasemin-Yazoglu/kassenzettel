# kassenzettel

## Über
Kassenzettel ist eine Web-App, die deine Lebensmittelausgaben für jeden Monat trackt. So bekommst du einen Überblick wie viel du jeden Monat für Lebensmittel ausgibst.

## Download
Clone dieses Repository, um die App zu benutzen.
Benutze hierfür:
`git clone [url]`
oder
`git clone [ssh]`

## Nutzung
Wechsle zuerst in das Verzeichnis.

### Dependencies Installieren
Führe `npm install` aus.

### App starten
Starte die App dann mit `npm run dev`.

### App aufrufen
Aufrufen kannst du die App auf deinem Webbrowser unter: `http://localhost:3000`.
